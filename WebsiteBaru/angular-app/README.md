# Angular App

Ini skeleton Angular. Langkah cepat:

1. `cd angular-app`
2. `npm install` (atau `pnpm install` / `yarn install`)
3. `npx ng serve --open`

Jika kamu ingin scaffold baru dari awal (direkomendasikan):

1. `npm install -g @angular/cli`
2. `ng new angular-app` di luar repo ini

Folder ini berisi definisi paket generik, gunakan untuk reference.
