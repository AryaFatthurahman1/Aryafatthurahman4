# Setup Host Lokal

File ini berisi mapping host lokal untuk Laragon.

## Isi yang perlu dipakai

Salin isi [hosts.txt](/D:/laragon/www/Mobile%20Computing%20Flutter/database/hosts.txt) ke file:

`C:\Windows\System32\drivers\etc\hosts`

## Setelah itu

1. Simpan file `hosts`
2. Buka Laragon
3. Restart `Apache`
4. Buka `http://mobile-computing-flutter.test/`

## Catatan

- Jangan pakai spasi di domain.
- Jangan pakai `%20` di hostname.
- Jika masih gagal, pakai `http://localhost/Mobile%20Computing%20Flutter/`
